# README

##Abstracts Parser App##

This application is for use by Alain Pinel Realtors Agents and Managers to import HTML files with (specifically the Mid-Peninsula Abstracts) and receive a parsed, processed .xls or .csv file with the relevant information. 

This appliclation also maintains a database of stored imported abstract information, so that abstract data can be searched, and display attendant results, via City, Date submitted or VPN number.
